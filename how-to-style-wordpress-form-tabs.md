## Classes

The available classes are as follows:

Class
Description
Where To Use

`wsf-steps`
Changes tabs into numeric steps
Tabs Wrapper Class

`wsf-steps-checks`
Show checkmarks on completed steps
Tabs Wrapper Class

`wsf-steps-success`
Change steps to green color
Tabs Wrapper Class

`wsf-vertical`
Change tabs or steps to a vertical format
Form Wrapper Class
